# Get MS2 Matches

A getter that returns the generated ms2 data in your `mass_data` object.

## Usage

``` r
get_ms2_matches(mass_data)
```

## Arguments

- mass_data:

  The object generated from
  [`ms2_ms1_compare()`](https://www.mums2.org/mums2/reference/ms2_ms1_compare.md).

## Value

a `data.frame` object containing ms2 data.

## Examples

   import_all_data(peak_table =
                   mums2::mums2_example("botryllus_pt_small.csv"),
                   metadata =
                   mums2::mums2_example("boryillus_metadata.csv"),
                   format = "None")

filtered_data <- data |>
   filter_peak_table(filter_mispicked_ions_params()) |>
   filter_peak_table(filter_cv_params(cv_threshold = 0.2)) |>
   filter_peak_table(filter_group_params(group_threshold = 0.1,
